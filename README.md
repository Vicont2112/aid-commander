## AID Commander — состояние проекта

---

### Что это

Тренажёр психологической устойчивости через первую помощь. Медицинские протоколы — не цель, а инструмент. Главное — выработать навык **не тормозить** в момент стресса. Ступор над телом и ступор перед трудным разговором — одна физиология. Тренировка одна.

Один HTML-файл. Работает на iOS и Android без установки. PWA — можно добавить на рабочий стол.

---

### Что внутри сейчас

**Combat Mode** — первый экран. Четыре состояния: Паника / Завис / Злость / Перегруз. Каждое — свой пошаговый протокол с дыханием. Физиологический вздох (6 сек) для острого состояния. Box Breathing 4-4-4-4 (16 сек) для перегруза. Счётчик возвратов контроля.

**27 сценариев экстренной помощи** в 7 категориях — Критические / Кровотечения / Дыхание / Травмы / Медицинские ЧС / Ожоги и поражения / Экстренные ситуации. Каждый сценарий — пошаговый алгоритм с таймером жгута, метрономом СЛР, кнопкой 112 и шпаргалкой «что говорить диспетчеру».

**Курс — 10 модулей** (Модуль 0–9): Победить ступор / Оценка обстановки / СЛР и AED / Кровотечения / Дыхательные пути / Травмы / Медицинские ЧС / Ожоги / Командир ситуации / Аптечка. Каждый модуль — 4 урока с полным текстом + практическое задание + финальный тест. Прогресс в localStorage.

**40 практических заданий** — конкретные действия которые можно сделать прямо сейчас. Не «повтори теорию» а «наложи жгут на бедро, засеки время».

**Тренажёр** — 50 вопросов, случайная выборка, правильный ответ подсвечивается.

**MIST отчёт** — форма для передачи раненого, копирование в буфер.

**История API** — свайп назад на iOS и кнопка назад на Android работают корректно. Плавающая кнопка ← Назад над навигацией.

### Нерешённые задачи

**Технические:**

- `sw.js` и `manifest.json` — файлы для PWA существуют в коде как ссылки, но сами файлы нужно создавать отдельно. Без них офлайн-режим не работает
- Языки UA / EN / PT — в v2 были реализованы, в v3 убраны для упрощения. Вернуть или нет — открытый вопрос

**Контент:**

- Модули курса написаны на базе текстов которые ты присылал (0–7). Модули 8 (Командир ситуации) и 9 (Аптечка) добавлены — но у тебя есть исходные MD-файлы только для части. Остальные модули ждут твоих текстов
- Combat Mode — протоколы сейчас написаны мной на основе Протокола Действия. Ты можешь уточнить формулировки

**Концептуальные:**

- Ежедневная практика — сейчас нет механики «5 минут в день». Стрик есть в Протоколе Действия, в AID Commander нет
- Связь между Combat Mode и сценариями — сейчас они живут отдельно. Логика «вышел из ступора → теперь действуй по протоколу» не выражена явно в интерфейсе
