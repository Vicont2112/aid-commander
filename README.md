## 🎯 Что это

Один HTML-файл (190 КБ) — приложение для iOS и Android. Работает без установки. PWA — можно добавить на рабочий стол.

**Философия:** Медицинские протоколы — не цель, а инструмент. Главное — выработать навык **не тормозить** в момент стресса. Ступор над телом и ступор перед трудным разговором — одна физиология. Тренировка одна.

## 📦 Что внутри

### 1️⃣ COMBAT MODE — 4 протокола управления состоянием

Первый экран. Четыре состояния → четыре протокола:

| Состояние | Протокол | Акроним | Физический якорь |
|-----------|----------|---------|------------------|
| 😤 **Паника** | 🛑 СТОП | **С-Т-О-П**<br>Стоп → Тело → Окружение → Поведение | Двойной вдох носом |
| 🧊 **Завис** | 🧊 Разморозка | **Ч-П-Н-Ш-Д**<br>Челюсть → Пальцы → Ноги → Шаг → Действие | Сжать/разжать кулаки 10 раз |
| 🔥 **Злость** | 🔥 Дистанция | **К-В-Ш-В**<br>Кулаки → Выдох → Шаг → Выбор | Раскрыть ладони |
| 🌀 **Перегруз** | ⏸️ Пауза | **С-Д-3-П**<br>Сядь → Дыши → 3 дела → Первое | Рука на живот |

**Техники дыхания:**
- **Физиологический вздох** (6 сек) — для острого состояния
- **Box Breathing 4-4-4-4** (16 сек) — для перегруза

**Фичи:**
- Счётчик возвратов контроля (сохраняется в localStorage)
- Анимация дыхания
- Тактильная отдача (vibration)

### 4️⃣ ПРАКТИЧЕСКИЕ ЗАДАНИЯ

~40 заданий — конкретные действия которые можно сделать **прямо сейчас**:

**Примеры:**
- «Наложи жгут на бедро поверх одежды. Засеки время. Цель: менее 30 секунд»
- «Встань и найди правильное положение для Геймлиха: кулак выше пупка, 5 имитационных толчков»
- «Отработай стабильное боковое положение на диване. 5 повторений. Цель: менее 20 секунд»
- «Прямо сейчас возьми свою аптечку и проверь: есть ли там жгут, гемостатик, перчатки?»

**Не «повтори теорию» — а «сделай здесь и сейчас».**

### 6️⃣ MIST ОТЧЁТ

Форма для передачи раненого эвакуационной группе (TCCC стандарт):

```
M — Mechanism (механизм травмы)
I — Injury (найденные повреждения)
S — Signs (жизненные показатели)
T — Treatment (оказанная помощь)
```

Копирование в буфер одним кликом.

## 🛠️ Технические детали

### Файлы
```
aid_commander_v3.html    — приложение (2160 строк, 190 КБ)
README.md                — этот файл
course_manifesto.md      — манифест курса
icon.svg                 — иконка PWA
manifest.json            — PWA манифест
sw.js                    — Service Worker (офлайн)
```

### PWA (Progressive Web App)
- ✅ `manifest.json` — описывает приложение
- ✅ `sw.js` — Service Worker для офлайн-режима
- ✅ `icon.svg` — иконка для рабочего стола

**Как работает офлайн:**
1. Первый запуск — файлы кэшируются
2. Следующие запуски — работают без интернета
3. Обновление кэша — при изменении версии

### История API
- ✅ Свайп назад на iOS
- ✅ Кнопка назад на Android
- ✅ Плавающая кнопка «← Назад» над навигацией

### localStorage
Сохраняется:
- Прогресс курса (модули, уроки, тесты)
- Счётчик возвратов контроля (Combat Mode)
- Настройки (если будут)
